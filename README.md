# 📻 2025版业余无线电台操作证刷题助手

一款专为业余无线电爱好者（HAM）打造的在线刷题工具，基于 CRAC 2025 年最新题库开发。

**🚀 在线访问地址：[http://ham_quiz.telysta.com/](http://ham_quiz.telysta.com/)**

---

## ✨ 项目特性

- **最新题库**：同步更新 2025 年 CRAC A/B/C 类最新完整题库。
- **全真模拟**：严格遵循官方组卷规则，包含单选、多选配置及倒计时功能。
- **智能抽题**：支持“随机模式”和“不重复模式（优先抽取未做题目）”。
- **错题特训**：自动记录错题，支持针对性强化练习，直到彻底掌握。
- **快速查题**：内置全量题库搜索引擎，支持按 ID 或关键词秒级检索。
- **响应式设计**：完美适配 PC 端与手机移动端。

## 🛠️ 技术实现

- **前端**：原生 HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript.
- **数据**：JSON 格式题库存储，支持离线运行。
- **部署**：GitHub Pages / 独立服务器部署。

## 📂 目录结构说明

- `index.html`: 页面主体结构
- `style.css`: 现代感 UI 样式设计
- `app.js`: 核心逻辑（含组卷算法、模拟计时、本地存储等）
- `bank_a.json / bank_b.json / bank_c.json`: 分类题库数据
- `images/`: 题目附图文件夹
